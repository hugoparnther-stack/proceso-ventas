# 🛒 Proceso de Ventas

Script en Python para practicar el uso de `*args` y `**kwargs`. 🐍

## ✨ ¿Qué hace?

- 📋 Guarda una lista de ventas (producto y precio).
- ➕ `calcular_total(*precios)`: suma cualquier cantidad de precios.
- 🧾 `registrar_compra(**datos)`: guarda datos del cliente (nombre, ciudad, método de pago, etc.).

## ▶️ Cómo usarlo

```bash
python sales.py
```
